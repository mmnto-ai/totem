## Lesson — Git diff headers wrap file paths containing spaces

**Tags:** architecture, curated
**Pattern:** (['"\/])\^?\+\+\+\s+b\\?\/
**Engine:** regex
**Scope:** **/\*.ts, **/_.js, \*\*/_.py, **/\*.sh, **/_.go, \*\*/_.rs
**Severity:** error

Git diff headers wrap paths with spaces in quotes (e.g., +++ "b/path"). Update logic to handle optional quotes before the 'b/' prefix to ensure files with spaces are processed correctly.
