## Lesson — For local LLM providers, 500 Internal Server Errors often

**Tags:** architecture, curated
**Pattern:** ^(?!.*(numCtx|context|VRAM|exhaustion))(?=.*500)(?=._Internal\s+Server\s+Error)._$
**Engine:** regex
**Scope:** **/llm/**/_.ts, **/providers/**/_.ts, !**/\*.test.ts
**Severity:\*\*\*\* error

Local LLM 500 errors often indicate VRAM or context exhaustion. Include specific guidance to adjust hardware-steering parameters like 'numCtx' in the error message to help users resolve failures immediately.
