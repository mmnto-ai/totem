## Lesson — Static top-level imports from heavy internal packages delay

**Tags:** performance, curated
**Pattern:** ^import\s+(?!type\s)(?!\s*\{[^}]*Error)._from\s+['"](?:../|@[\w-]+/)
**Engine:** regex
**Scope:** packages/cli/src/commands/\*\*/_.ts, !**/\*.test.ts
**Severity:\*\*\*\* warning

Static top-level imports of internal packages delay CLI startup. Use dynamic imports inside command functions to ensure dependencies are only loaded when needed. (Error class imports are exempt.)
