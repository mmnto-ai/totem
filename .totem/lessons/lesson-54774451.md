## Lesson — When detecting markers in user-authored files, use

**Tags:** style, curated
**Pattern:** \.(includes|indexOf|match)\(\s*['"]marker['"]
**Engine:** regex
**Scope:** \*\*/*.ts, **/\*.js, **/_.tsx, \*\*/_.jsx, !**/\*.test.ts
**Severity:\*\* warning

Use line-start regex (/^marker/m) instead of simple string search for markers to prevent corruption if the marker string appears within the content body.
