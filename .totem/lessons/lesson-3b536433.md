## Lesson — Prefer Explicit Metadata Tokens from LLMs Over Heuristics

**Tags:** security, curated
**Pattern:** \.split\(['"/](?:\\n|\\r\\n)['"/]\)\s*\[\s*0\s*\]
**Engine:** regex
**Scope:** **/*.ts, **/*.js, !**/*.test.ts
**Severity:** error

Avoid heuristic truncation of the first line (e.g., .split('\n')[0]). Instead, request explicit metadata tokens from the LLM (like 'Heading:') and sanitize them to remove markdown artifacts or prefixes.
