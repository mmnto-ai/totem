## Lesson — Provide a dummy fallback API key (e.g., 'local-only')

**Tags:** architecture, curated
**Pattern:** \bapiKey:\s*process\.env\.[A-Z0-9_]+(?!\s*(\|\||\?\?))
**Engine:** regex
**Scope:** **/\*.ts, **/_.tsx, \*\*/_.js, **/\*.jsx, !**/\*.test.ts
**Severity:** error

Provide a dummy fallback API key (e.g., || 'local-only') when using environment variables to ensure local OpenAI-compatible endpoints bypass SDK validation.
