## Lesson — Anchor CLI artifacts to configRoot

**Tags:** cli, monorepo, paths
**Scope:** packages/cli/src/commands/**/*.ts, !**/*.test.*, !**/*.spec.*

Resolve internal tool directories (like .totem/) relative to the configuration file's location rather than the current working directory. This prevents path resolution failures in monorepos when commands are executed from nested subpackages where cwd differs from the project root.
